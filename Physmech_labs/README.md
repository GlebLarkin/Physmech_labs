## Структура папок
Для каждой работы существует своя папка с соответствующим номером XX. 
В этой папке находятся подпапки с работами разных авторов (в формате Фамилия_И). 

*Работы разных авторов могут иметь различную структуру.*

## Как добавить свою работу?

- Если добавляете **новую** лабу — создайте папку в корне репозитория с названием `XX`, где `XX` — номер работы, и добавьте номер лабораторной в корневой `README.md`.
- Если лаба **уже существует** — добавляйте файлы в соответствующую папку.

> [!IMPORTANT]
> Также обязательно добавьте pdf своей работы в папку pdf. Если такой папки нет, создайте ее.

**Что нужно добавить:**
- Скомпилированный отчёт (`.pdf`)
- Исходник отчёта (`.tex`)
- Сырые данные
- Код для обработки данных *(по возможности)*

> [!CAUTION]
> Файл `.tex` должен компилироваться без нестандартных библиотек и неучтённых локальных файлов.

> *Опциональная, но большая просьба* — добавьте в свою папку `README.md` с замечаниями, которые были сделаны во время сдачи работы.
