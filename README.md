# Physmech_labs
*Лабораторные работы по физической механике, ФАКТ МФТИ, 3-4 курсы.*

>[!note]
>**Цель этого репозитория - не разместить готовые отчеты, а дать студентам референс для сравнения результатов.**

Замечания и предложения приветствуются.

## Список лабораторных работ, которые тут можно найти:
- 20

*Также в папке materials есть лабник. Можете добавлять туда полезные материалы.*

## Структура папок
Для каждой работы существует своя папка с соответствующим номером XX. 
В этой папке находятся подпапки с работами разных авторов (в формате Фамилия_И). 

*Работы разных авторов могут иметь различную структуру.*

---

## Как добавить свои работы
### 1. Сделайте Fork и клонируйте его к себе

### 2. Создайте новую ветку

>[!warning]
> *Не работайте в main.*

Рекомендуемый формат имени ветки: lab-XX-username

### 4. Добавьте свою работу
**Если добавляете новую лабораторную работу, создавайте, пожалуйста, новую папку в корне с названием XX, где XX - номер работы. Иначе добавляйте в уже существующую.**

Добавляйте, пожалуйста, следующее:
- скомпилированный отчет
- .tex файл
- сырые данные
- по возможности код для обработки.

**Файл .tex должен компилироваться без подгрузки нестандартных библиотек и неучтенных локальных файлов.**
  
_*Большая, но опциональная просьба* - добавляйте, пожалуйста, в свои папки локальный readme.md с указанием замечаний к отчету во время сдачи._

>[!warning]
> **Очевидно, но не изменяйте и не удаляйте чужие работы (Я очень надеюсь, что вы не сможете этого сделать)**

После этого сделайте коммит и отправьте изменения в свой fork.

### 5. Создайте Pull Request

После отправки ваш Pull Request будет проверен и при отсутствии замечаний влит в основную ветку.
