# Комментарии после сдачи:
*Отчет вообще почти не смотрят, да и смотреть там нечего. Основной уклон на сдаче идет на решение задач.*

Сначала нужно было вывести критерий Релея для устойчивости течения Куэтта в невязком случае.
Это делается просто - нужно рассмотреть изменение суммарный силы (градиент давлений и центробежная), 
действующей на элемент жидкости, разложиться в ряд Тейлора и аккуратно все сократить. 
Выходит, что $dF = -2 \rho \Omega / r d(\Omega r^2)/dr dr$, те при положительных значениях $-d(\Omega r^2)/dr$ есть возвращаюащя сила,
те течение устойчиво.

Затем мы перешли на рассмотрение вязкого течения - нужно было оценить максимальную относительную флуктуацию давлений, 
которая еще не приведет к неустойчивости, то есть найти флукциация давлений, которая может быть поглащена вязкими силами.

По определению $\eta, Re$:
$ \delta P ~ \eta d(\Omega r)/dr ~ \frac{\rho \Omega r^2}{Re} d(\Omega r)/dr ~ P / Re$
То есть $\delta P / P ~ 1 / Re$. Преподавателя этот ответ очень порадовал, и он поставил всем 10.
